1. cloning
```bash
git clone https://github.com/Dzyfhuba/laravel-and-flutter-developer.git test-hafidz
cd test-hafidz
cp .env.example .env
```

2. setup environtment variable in ```.env```

3. serving
```bash
composer install
npm i
npm run build
php artisan migrate
php artisan db:seed
php artisan serve
```
